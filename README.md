## ğŸš€ Getting Started

### Prerequisites

- A modern web browser
- Git (for cloning and version control)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/amolahinge/amolahinge.github.io.git
   ```

2. Navigate to the project directory:
   ```bash
   cd amolahinge.github.io
   ```

3. Open `index.html` in your web browser or use a local server:
   ```bash
   # Using Python 3
   python -m http.server 8000
   
   # Using Node.js (if you have http-server installed)
   npx http-server
   ```

4. Visit `http://localhost:8000` in your browser

## ğŸ¨ Customization

### Colors
The main color scheme can be customized by modifying the CSS variables:
- Primary: `#3498db` (Blue)
- Secondary: `#2c3e50` (Dark Blue)
- Accent: `#f39c12` (Orange)

### Content
Update the following files to customize content:
- `index.html`: Personal information, projects, and contact details
- `styles.css`: Styling and layout adjustments
- `script.js`: Interactive functionality

### Adding Projects
To add new projects, duplicate the project card structure in `index.html`:

```html
<div class="project-card">
    <div class="project-image">
        <div class="placeholder-image">
            <i class="fas fa-your-icon"></i>
        </div>
    </div>
    <div class="project-content">
        <h3>Your Project Name</h3>
        <p>Project description...</p>
        <div class="project-tech">
            <span>Technology 1</span>
            <span>Technology 2</span>
        </div>
        <div class="project-links">
            <a href="#" class="btn btn-small">Live Demo</a>
            <a href="#" class="btn btn-small btn-outline">GitHub</a>
        </div>
    </div>
</div>
```
